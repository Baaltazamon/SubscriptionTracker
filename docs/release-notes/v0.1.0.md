# Subscription Tracker v0.1.0

## Highlights

- WPF desktop MVP with a custom shell and dark/light themes
- Local-first storage with SQLite
- Subscription, category, history, calendar, and analytics screens
- Backup and restore for the local database
- Windows reminders, toast notifications, and launch with Windows
- English and Russian localization
- Excel export and dashboard insights for rarely used subscriptions

## Packaging

- Windows publish bundle
- ZIP archive for manual download and launch

## Notes

- Currency rates are currently offline and fixed
- LiveCharts2 integration may emit `NU1701` warnings during restore/build, but the application builds and runs
