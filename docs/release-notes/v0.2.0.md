# Subscription Tracker v0.2.0

## Highlights

- Import workflow expanded from a simple file load into a controlled product flow
- Manual exchange rates added to settings and used across dashboard, analytics, and projections
- Import preview now supports selective apply before changes hit the database
- Last import can be rolled back from stored import session snapshots
- Recent import sessions are visible directly in the subscriptions screen

## Import Workflow

- `CSV` and `XLSX` subscription import
- Downloadable import templates
- Preview before apply
- Row-by-row selective import
- Upsert by subscription name
- Auto-creation of missing categories
- Undo last import
- Import session history log

## Validation

- `dotnet build SubscriptionTracker.sln`
- `dotnet test SubscriptionTracker.Tests\SubscriptionTracker.Tests.csproj --no-build`
- `dotnet publish SubscriptionTracker.Wpf\SubscriptionTracker.Wpf.csproj -c Release -o artifacts\publish\wpf`
- `.\installer\Create-InstallerBundle.ps1 -Version v0.2.0`

- Currency rates remain offline and user-managed
- Release packaging remains ZIP plus per-user installer bundle
- Signed installer / MSIX is still a later step
