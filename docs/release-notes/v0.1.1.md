# Subscription Tracker v0.1.1

## Highlights

- Clean `v0.1.1` stabilization pass with `0` build warnings
- Expanded automated smoke coverage for release-critical flows
- Isolated first-run support via `SUBSCRIPTION_TRACKER_APPDATA_DIR`
- Per-user installer bundle for Windows distribution

## Packaging

- Windows publish bundle
- Portable ZIP archive
- Per-user installer bundle:
  - `SubscriptionTracker-v0.1.1-win-x64-installer.zip`
  - installs to `%LocalAppData%\Programs\SubscriptionTracker`
  - creates a Start Menu shortcut
  - supports uninstall and optional user-data cleanup

## Validation

- `dotnet build SubscriptionTracker.sln`
- `dotnet test SubscriptionTracker.Tests\SubscriptionTracker.Tests.csproj --no-build`
- `dotnet publish SubscriptionTracker.Wpf\SubscriptionTracker.Wpf.csproj -c Release -o artifacts\publish\wpf`
- sandbox install/uninstall verification for the installer bundle

## Notes

- Currency rates are currently offline and fixed
- Manual visual smoke checks are still recommended for theme, dialog, and toast rendering on the target machine
- Signed MSIX packaging is not included in this release; the current distribution path is ZIP plus installer bundle
